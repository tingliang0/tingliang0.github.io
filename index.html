搭建静态博客

发表于 2016-11-08

本文记录了使用hexo、NexT、github搭建过程。

hexo

Hexo 是一个基于nodejs的静态博客网站生成器，作者是来自台湾的 Tommy Chen。特点：简单易用。

安装

假设已经安装好nodejs后，安装hexo非常简单。

测试是否创建成功

hexo创建的项目默认的访问地址是: http://127.0.0.1:4000/，因此只需要启动hexo项目，访问这个路径就知道是否创建成功了。

创建文章

命令成功执行后，默认会在source/_posts/目录下，生成一个hello.md，后续文章的内容都写在这个markdown文章里。如果想预览新加的文章，在浏览器里重新刷新页面即可。

NexT

Hexo创建的项目，默认使用的主题是landscape，而本人更喜欢NexT这个简洁明了的主题。下面演示如何在Hexo项目里替换主题。

安装

先把NexT这个主题从GitHub上拉到本地项目里

替换

项目根目录下的_config.yml记录了整个项目的配置信息。与主题相关的是theme字段。因此把theme的值从landscape替换成next即可完成整个主题的替换工作。

语言

_config.yml与语言相关的配置项是language，如果想使用简体中文，值配成zh-Hans即可。

GitHub

Github最终会作为我们博客的运行的服务器，由于Github只支持静态文件，因此在发布前要先将我们博客的内容转换为静态文件。

转换
