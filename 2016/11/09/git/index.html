发表于 2016-11-09

本文记录了日常开发中常用的git命令，方便查找，不定期更新。

git submodule

现在是一个多人协作的时代了，开发中常常要用到别人开发的工具和库。但是很多时候，我们只是想用这个库，而不想将这个库纳入自己的版本管理里。我们希望库的作者继续维护这个库，作为使用者的我们只需简单基本即可更新到库的最新修改。git submodule就是为了解决这个问题出现的。下面演示怎样通过命令来达到上面目的。
